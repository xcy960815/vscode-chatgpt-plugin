## 一、代码重构与架构优化

> 目标：将过时的架构彻底现代化，提升可维护性

### 1.1 依赖清理 ✅

| 任务                                         | 状态 |
| -------------------------------------------- | ---- |
| 安装官方 `openai` SDK (`npm install openai`) | ✅   |
| 移除 `isomorphic-fetch`、`node-fetch`        | ✅   |
| 移除 `eventsource-parser`（手写 SSE 解析）   | ✅   |
| 移除 `gpt3-tokenizer`、`gpt-tokenizer`       | ✅   |
| 移除 `p-timeout`（超时控制）                 | ✅   |
| 移除 `puppeteer` / `puppeteer-extra` 全系列  | ✅   |
| 移除 `Keyv`、`QuickLRU`（对话缓存）          | ✅   |
| 移除 `rimraf` 过时 API 警告                  | ✅   |

### 1.2 核心架构重构 ✅

| 任务                                             | 状态 |
| ------------------------------------------------ | ---- |
| 创建 `src/openai-service.ts`，封装官方 SDK 调用  | ✅   |

| 使用原生 `AbortController` 支持中断生成          | ✅   |

| 删除 `src/text-model-api.ts`                     | ✅   |
| 删除 `src/utils.ts`（`fetchSSE` 已无用）         | ✅   |
| 创建 `src/config.ts`，统一管理所有配置读取       | ✅   |
| 创建 `src/commands.ts`，解耦命令注册逻辑         | ✅   |
| 创建 `src/i18n.ts`，抽离国际化工具类             | ✅   |
| `ChatgptViewProvider` 精简，移除"上帝类"设计     | ✅   |
| 删除 `isGptModel`、`isTextModel` 冗余判断        | ✅   |

## 三、核心功能增强

### 3.1 上下文感知（Context Awareness）✅

| 任务                                               | 状态 |
| -------------------------------------------------- | ---- |
| "当前文件"按钮：一键将当前打开文件附加到对话上下文 | ✅   |

| 右键菜单增加通用"询问选中内容"入口                   | ⏳   |
| 支持多段选中合并后一起发送给 AI                      | ⏳   |
